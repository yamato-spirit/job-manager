# Job Manager - 就活状況管理アプリケーション

## 📖 概要
就職活動における数十社の選考状況を一元管理するWebアプリケーションです。
既存のエクセル管理やPC専用ツールで生じていた「移動中にスマホで確認・編集しづらい」「優先順位の並び替えが手間」という課題を解決するために開発しました。

## 🚀 こだわりの機能・技術
採用担当者様に向けて、特に注力した技術ポイントを記載します。

### 1. スマホ最適化（Mobile First UI）
* **課題:** 既存ツールはPC表示がメインで、外出先での確認が困難だった。
* **解決:** レスポンシブデザインを採用。Flexboxによるレイアウト制御、`overflow-wrap` による文字の折り返し制御を行い、スマホの狭い画面でもストレスなく閲覧・編集が可能です。

### 2. 直感的な優先順位管理 (Ajax / SortableJS)
* **課題:** 志望度の変化や締め切り順の入れ替え作業が煩雑だった。
* **解決:** `SortableJS` を導入し、ドラッグ＆ドロップによる直感的な並び替えを実現。JavaScriptの `fetch` APIを用いて非同期で順序データを送信するため、画面リロードなしでスムーズに保存されます。

### 3. ユーザー体験を損なわない状態維持
* **課題:** 詳細画面から一覧に戻った際、ソート順がリセットされてしまう。
* **解決:** ControllerとThymeleaf間でソート状態（昇順・降順・手動）をバケツリレー方式でパラメータとして引き継ぐ設計により、画面遷移後もユーザーの意図した表示順序を維持します。

### 4. 24時間365日の安定稼働
* **工夫:** Render等のPaaS環境でのスリープを防ぐため、UptimeRobotと連携した専用のエンドポイント（`/ping`）を実装。いつでも即座にアクセスできる環境を構築しました。

## 🛠 使用技術
* **言語:** Java 21
* **フレームワーク:** Spring Boot 3 (Spring Data JPA, Spring Web)
* **テンプレートエンジン:** Thymeleaf
* **フロントエンド:** HTML5, CSS3, JavaScript (ES6+), SortableJS
* **データベース:** MySQL (開発環境) / PostgreSQL (本番環境)
* **インフラ/ツール:** Docker, Render, UptimeRobot

## 🏗 設計・ディレクトリ構成
MVCアーキテクチャを採用し、可読性と保守性を意識した構成にしています。

* `Controller`: 画面遷移とデータの受け渡し制御（状態維持ロジック含む）
* `Repository`: Spring Data JPAによるデータベース操作の抽象化
* `Entity`: 選考状況データの定義
